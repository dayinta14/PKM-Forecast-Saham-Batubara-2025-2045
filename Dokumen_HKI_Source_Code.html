1.3 Akuisisi dan Pembangkitan Dataset

Dataset harga saham sektor batubara Indonesia dibangkitkan secara deterministik menggunakan model Geometric Brownian Motion (GBM) untuk empat emiten utama: ADRO, PTBA, INDY, dan ITMG. Pembangkitan memperhitungkan drift tahunan, volatilitas annualized, efek booming batubara (2022), dan efek crash COVID-19 (2020). Log return harian dihitung sebagai transformasi logaritmik natural dari rasio harga berturut-turut. 
np.random.seed(2024)

def buat_seri_saham(n, harga_awal, drift_tahunan, vol_tahunan,
                    booming_idx=None, booming_mag=0.0,
                    crash_idx=None, crash_mag=0.0):

    dt    = 1 / 252
    mu    = drift_tahunan / 252
    sigma = vol_tahunan / np.sqrt(252)

    log_ret = np.random.normal(mu - 0.5 * sigma**2, sigma, n)

    if booming_idx is not None:
        log_ret[booming_idx:booming_idx+60] += booming_mag / 60
    if crash_idx is not None:
        log_ret[crash_idx:crash_idx+40] += crash_mag / 40

    harga = harga_awal * np.exp(np.cumsum(log_ret))
    return harga

1.6 Uji Stasioneritas

Uji stasioneritas formal dilaksanakan menggunakan dua metode komplementer: Augmented Dickey-Fuller (ADF) Test untuk menguji hipotesis nol bahwa seri memiliki unit root, dan KPSS Test sebagai konfirmasi silang yang menguji hipotesis nol bahwa seri bersifat stasioner. Pengujian dilakukan pada tiga bentuk data: level, differencing orde pertama (Δ¹), dan log return. 
def uji_stasioneritas(series, label):

    adf_res = adfuller(series.dropna(), autolag="AIC")
    adf_stat, adf_p = adf_res[0], adf_res[1]
    adf_cv1, adf_cv5 = adf_res[4]["1%"], adf_res[4]["5%"]

    kpss_res = kpss(series.dropna(), regression="c", nlags="auto")
    kpss_stat, kpss_p = kpss_res[0], kpss_res[1]

    return {
        "Series":            label,
        "ADF Stat":          f"{adf_stat:.4f}",
        "ADF p-value":       f"{adf_p:.6f}",
        "ADF CV 1%":         f"{adf_cv1:.4f}",
        "ADF CV 5%":         f"{adf_cv5:.4f}",
        "ADF Result":        "Stationary" if adf_p < 0.05 else "Non-Stationary",
        "KPSS Stat":         f"{kpss_stat:.4f}",
        "KPSS p-value":      f"{'>{:.2f}'.format(kpss_p) if kpss_p > 0.1 else '<0.05'}",
        "KPSS Result":       "Non-Stationary" if kpss_stat > 0.463 else "Stationary",
        "Integration Order": "",
    }

stat_rows = []
for ticker in ticker_list:
    r_level = uji_stasioneritas(df[ticker], f"{ticker} Level")
    r_diff1 = uji_stasioneritas(df[ticker].diff().dropna(), f"{ticker} Δ¹")
    r_logrt = uji_stasioneritas(df[f"ret_{ticker}"].dropna(), f"{ticker} Log Return")

    if r_level["ADF Result"] == "Non-Stationary" and r_diff1["ADF Result"] == "Stationary":
        r_level["Integration Order"] = "I(1)"
    elif r_level["ADF Result"] == "Stationary":
        r_level["Integration Order"] = "I(0)"

    stat_rows.extend([r_level, r_diff1, r_logrt])

2.3 Grid Search Orde ARIMA (p, d, q)

Pencarian orde optimal ARIMA dilakukan melalui grid search exhaustive. Seluruh kombinasi p ∈ {0,1,2,3}, d = 1, dan q ∈ {0,1,2,3} dievaluasi untuk setiap emiten. Orde differencing d ditetapkan 1 berdasarkan hasil uji stasioneritas (BAB 1) yang menunjukkan integrasi I(1). Seleksi model terbaik berdasarkan Akaike Information Criterion (AIC) minimum. 
def grid_search_arima(series, p_range=range(4), d_vals=(1,), q_range=range(4)):
    """
    Cari kombinasi (p,d,q) terbaik berdasarkan AIC.
    Mengembalikan DataFrame hasil lengkap, diurutkan AIC ascending.
    """
    results = []
    for d in d_vals:
        for p, q in itertools.product(p_range, q_range):
            if p == 0 and q == 0:
                continue
            try:
                mdl = SARIMAX(
                    series, order=(p, d, q), trend="n",
                    enforce_stationarity=False,
                    enforce_invertibility=False
                ).fit(disp=False, method="lbfgs", maxiter=200)
                results.append({
                    "p": p, "d": d, "q": q,
                    "AIC":  round(mdl.aic, 2),
                    "BIC":  round(mdl.bic, 2),
                    "HQIC": round(mdl.hqic, 2),
                    "LogL": round(mdl.llf,  2),
                })
            except Exception:
                pass
    return pd.DataFrame(results).sort_values("AIC").reset_index(drop=True)

grid_results = {}
best_orders  = {}

2.5 Rolling Forecast 1-Step Ahead

Evaluasi out-of-sample dilakukan menggunakan metode rolling forecast 1-step ahead yang lebih realistis dibandingkan multi-step langsung. Pada setiap langkah, model di-refit dengan menambahkan observasi aktual terbaru ke data training, kemudian memprediksi satu langkah ke depan. Metode ini menghasilkan evaluasi performa yang tidak bias. 
def rolling_forecast_arima(train_series, test_series, order):
    """
    1-step ahead rolling forecast:
    model diupdate setiap hari dengan menambahkan observasi aktual.
    """
    history   = list(train_series)
    forecasts = []
    for t in range(len(test_series)):
        mdl  = SARIMAX(
            history, order=order, trend="n",
            enforce_stationarity=False,
            enforce_invertibility=False
        ).fit(disp=False, method="lbfgs", maxiter=200)
        fc   = mdl.forecast(steps=1)[0]
        forecasts.append(fc)
        history.append(test_series.iloc[t])
    return np.array(forecasts)

        df_train[ticker].dropna(), df_test[ticker], best_orders[ticker]
    )
    arima_forecast[ticker] = pd.Series(fc, index=df_test.index[:len(fc)])

2.6 Evaluasi Model ARIMA

Evaluasi performa menggunakan empat metrik standar forecasting: MAE (Mean Absolute Error), RMSE (Root Mean Squared Error), MAPE (Mean Absolute Percentage Error), dan R² (Coefficient of Determination). 
def hitung_metrik(aktual, prediksi, label="Model"):
    mae  = mean_absolute_error(aktual, prediksi)
    rmse = np.sqrt(mean_squared_error(aktual, prediksi))
    mape = np.mean(np.abs((aktual - prediksi) / aktual)) * 100
    r2   = r2_score(aktual, prediksi)
    return {
        "Model": label,
        "MAE":   round(mae,  4),
        "RMSE":  round(rmse, 4),
        "MAPE (%)": round(mape, 4),
        "R²":    round(r2,   6),
    }

arima_metrics = {}
for ticker in ticker_list:
    aktual = df_test[ticker].values
    fc     = arima_forecast[ticker].values

    m["Emiten"] = ticker

3.2 Normalisasi Data dan Pembentukan Sequence

Data time series ditransformasi ke format supervised learning melalui dua tahap: (1) normalisasi menggunakan MinMaxScaler ke rentang [-1, 1] untuk menstabilkan konvergensi gradient LSTM, dan (2) pembentukan windowed sequences dengan jendela 30 hari (satu bulan perdagangan). Setiap sampel input memiliki dimensi (30, 1) dan memprediksi satu nilai output. 
WINDOW        = 30     # jumlah timestep input (look-back period)
LSTM_UNITS    = 64     # neuron per layer LSTM
DROPOUT_RATE  = 0.2    # dropout regularisasi
LEARNING_RATE = 1e-3   # Adam initial LR
BATCH_SIZE    = 32
EPOCHS        = 100    # max epoch (Early Stopping)
PATIENCE      = 15     # Early Stopping patience

def buat_sequences(data: np.ndarray, window: int):
    """
    Ubah array 1D menjadi pasangan (X, y) untuk LSTM.
    X shape : (n, window, 1)
    y shape : (n,)
    """
    X, y = [], []
    for i in range(window, len(data)):
        X.append(data[i-window:i])
        y.append(data[i])
    return np.array(X)[..., np.newaxis], np.array(y)

def siapkan_data_lstm(series_train, series_test, window):
    """Scale + buat sequences train & test."""
    scaler = MinMaxScaler(feature_range=(-1, 1))
    train_sc = scaler.fit_transform(series_train.values.reshape(-1, 1)).flatten()
    test_sc  = scaler.transform(series_test.values.reshape(-1, 1)).flatten()

    X_tr, y_tr = buat_sequences(train_sc, window)
    X_te, y_te = buat_sequences(test_sc,  window)
    return X_tr, y_tr, X_te, y_te, scaler
Input: Seri harga training dan testing per emiten.
Output: Tensor X_train (n, 30, 1) dan y_train (n,) per emiten.

3.3 Arsitektur Model LSTM

Arsitektur LSTM dual-layer dirancang dengan regularisasi berlapis: Dropout (0.2) pada setiap layer LSTM, L2 regularization (λ=10⁻⁴) pada kernel, serta Dense layer intermediate (32 unit, ReLU). Arsitektur yang sama digunakan untuk LSTM murni dan LSTM residual, memastikan konsistensi perbandingan. 
def bangun_lstm(window, lstm_units=64, dropout=0.2, lr=1e-3):
    """
    Membangun arsitektur LSTM dua layer.
    Dipakai baik untuk LSTM murni maupun LSTM pada residual ARIMA.
    """
    model = Sequential([
        Input(shape=(window, 1)),
        LSTM(lstm_units, return_sequences=True,
             kernel_regularizer=l2(1e-4)),
        Dropout(dropout),
        LSTM(lstm_units, return_sequences=False,
             kernel_regularizer=l2(1e-4)),
        Dropout(dropout),
        Dense(32, activation="relu"),
        Dense(1)
    ])
    model.compile(
        optimizer=Adam(learning_rate=lr),
        loss="mse",
        metrics=["mae"]
    )
    return model
Arsitektur: Input(30,1) → LSTM(64) → Dropout(0.2) → LSTM(64) → Dropout(0.2) → Dense(32,ReLU) → Dense(1)

3.5 Training LSTM pada Residual ARIMA

Model LSTM kedua dilatih khusus pada residual ARIMA — yaitu komponen non-linear yang tidak berhasil ditangkap oleh model statistik. Input berupa sequence residual ARIMA, dan output berupa prediksi residual satu langkah ke depan. Dengan demikian, LSTM belajar memodelkan pola non-linear yang tersisa setelah pemodelan linear ARIMA. 
def siapkan_data_residual(resid_train, resid_test, window):
    """Scale + buat sequences dari residual ARIMA."""
    scaler   = MinMaxScaler(feature_range=(-1, 1))
    train_sc = scaler.fit_transform(resid_train.values.reshape(-1,1)).flatten()
    X_tr, y_tr = buat_sequences(train_sc, window)
    return X_tr, y_tr, scaler

BAB 4. HYBRID STATISTICS AND MACHINE LEARNING METHOD

Bab ini mendokumentasikan implementasi integrasi model ARIMA (statistik) dan ANN-LSTM (machine learning) dalam arsitektur hybrid. Pendekatan hybrid ini merupakan inti inovasi sistem, dimana kedua paradigma pemodelan digabungkan secara sinergis melalui mekanisme residual learning untuk menghasilkan prediksi yang lebih akurat dibandingkan masing-masing model secara individu.

4.1 Integrasi ARIMA dan ANN-LSTM

Integrasi dilakukan melalui pendekatan dekomposisi sinyal: ARIMA menangkap komponen linear terlebih dahulu, kemudian LSTM memprediksi residual (komponen non-linear yang tersisa). Prediksi akhir diperoleh dari penjumlahan kedua komponen.
ŷhybrid(t) = ŷARIMA(t) + ŷLSTM(εARIMA(t))

dimana εARIMA(t) = X(t) − ŷARIMA(t) adalah residual model ARIMA.

4.2 Alur Sistem Hybrid

Proses hybrid dilaksanakan melalui empat langkah terintegrasi yang menghubungkan seluruh modul dalam pipeline:
LangkahProsesModul
1ARIMA menangkap komponen linear → menghasilkan prediksi ŷ_ARIMA(t) dan residual ε(t)BAB 2
2LSTM dilatih pada sequence residual ARIMA → belajar pola non-linearBAB 3
3LSTM memprediksi residual secara rekursif → menghasilkan ŷ_LSTM(ε(t))BAB 4
4Kombinasi: ŷ_hybrid(t) = ŷ_ARIMA(t) + ŷ_LSTM(ε(t))BAB 4

4.3 Implementasi Forecast Hybrid

Fungsi forecast_lstm_residual melakukan prediksi residual secara iteratif (recursive forecast). Pada setiap langkah, window terakhir dari buffer residual digunakan sebagai input LSTM, dan prediksi ditambahkan ke buffer untuk langkah berikutnya. Hasil prediksi residual kemudian dijumlahkan dengan forecast ARIMA. 
def forecast_lstm_residual(model, scaler, resid_train, n_steps, window):
    """
    Forecast n_steps ke depan secara iteratif menggunakan window
    terakhir residual. Setiap prediksi ditambahkan ke buffer
    untuk prediksi berikutnya (recursive forecast).
    """
    train_sc = scaler.transform(
        resid_train.values[-window:].reshape(-1, 1)
    ).flatten()
    buffer = list(train_sc)
    preds  = []

    for _ in range(n_steps):
        x_input  = np.array(buffer[-window:]).reshape(1, window, 1)
        pred_sc  = model.predict(x_input, verbose=0)[0, 0]
        pred_inv = scaler.inverse_transform([[pred_sc]])[0, 0]
        preds.append(pred_inv)
        buffer.append(pred_sc)

    return np.array(preds)

4.5 Forecast Jangka Panjang 2025–2045

Proyeksi jangka panjang 21 tahun ke depan (2025–2045) dilakukan menggunakan recursive multi-step forecast. ARIMA di-fit pada data penuh (train + test) dan menghasilkan forecast iteratif beserta interval kepercayaan 95%. LSTM residual memprediksi komponen non-linear secara rekursif. Kedua komponen dijumlahkan untuk menghasilkan prediksi hybrid jangka panjang. 
def forecast_arima_longterm(series_full, order, n_steps):
    """Forecast n_steps ke depan dari model ARIMA pada data penuh."""
    mdl = SARIMAX(
        series_full, order=order, trend="n",
        enforce_stationarity=False, enforce_invertibility=False
    ).fit(disp=False, method="lbfgs", maxiter=300)
    fc_obj  = mdl.get_forecast(steps=n_steps)
    fc_mean = fc_obj.predicted_mean
    ci      = fc_obj.conf_int(alpha=0.05)
    return fc_mean.values, ci.values[:, 0], ci.values[:, 1]
